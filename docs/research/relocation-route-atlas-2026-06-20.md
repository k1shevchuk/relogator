# Relogator: черновой атлас разветвленных маршрутов

Дата: 2026-06-20.

Статус: продуктовый черновик для наполнения базы данных и постановки задач агентам-сборщикам. Это не юридическая консультация и не публичная правовая база. Перед показом пользователю каждая ветка должна получить официальный источник, дату проверки, применимость к гражданам РФ и статус `reviewed`, `needs_review`, `stale` или `reference_only`.

ИИ использовался только как вспомогательный инструмент структурирования. Источником истины должны быть официальные сайты консульств, миграционных ведомств, законов, порталов госуслуг, статистики и проверенных партнеров.

## Единая модель веток

| Ветка         | Цель пользователя                                      | Что должно спрашиваться в анкете                                                            | Типовые блокеры                                                                             | Что открывает маршрут                                                         |
| ------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| `short_entry` | быстро въехать, разведка, 1-3 месяца                   | паспорт, срок действия, прошлые отказы, цель поездки, адрес, страховка, обратный маршрут    | нет визы/разрешения, короткий паспорт, запрет на въезд, неподтвержденная цель               | действующий паспорт, понятная цель, адрес, страховка, деньги на первый период |
| `medium_stay` | 3-12 месяцев без немедленного ПМЖ                      | допустимый срок пребывания, регистрация/адрес, доход, семья, школа, питомцы                 | пользователь хочет жить дольше короткого въезда без основания                               | временное проживание, продление, учеба, работа, семья, удаленная работа       |
| `residence`   | ВНЖ, временное проживание, долгий статус               | основание, доход, работа, бизнес, семья, учеба, справки, переводы                           | нет основания, нет денег/адреса, документы без перевода, просроченная подача                | подтвержденное основание, комплект документов, запись, адрес                  |
| `remote_work` | удаленная работа на иностранного работодателя/клиентов | страна работодателя, договор, доход, выписки, профессия, налоги                             | туристический въезд принят за право работать, доход не подтвержден, работа на местный рынок | отдельная виза удаленной работы, foreign income, подтвержденные выписки       |
| `local_work`  | работа у местного работодателя                         | оффер, договор, профессия, диплом, работодатель готов участвовать                           | нет работодателя, работа без разрешения, диплом не признается                               | оффер, work permit/single permit, recognized sponsor, квалификация            |
| `business`    | ИП, компания, инвестиции, самозанятость                | бизнес уже есть, готовность открыть компанию, капитал, клиенты, адрес, налоговая готовность | бизнес только на словах, нет капитала, нет права работать                                   | компания, инвестиционный план, self-employment route, лицензия                |
| `family`      | переезд с супругом, детьми, родителями                 | состав семьи, документы о браке/рождении, согласия, школа/сад, доход спонсора               | нет статуса у главного заявителя, не готовы документы родства                               | основной заявитель получает статус, документы родства, dependants             |
| `study`       | учеба как основание                                    | зачисление, срок учебы, оплата, язык, жилье                                                 | нет admission, учеба не дает нужный срок/работу                                             | admission letter, student visa/residence, средства                            |
| `pets`        | переезд с животным                                     | вид животного, чип, прививки, ветпаспорт, сертификат, маршрут                               | нет чипа/вакцины/сертификата, срок подготовки меньше требований                             | ветсертификат, чип, вакцина, правила ввоза страны                             |

Шкала сложности:

- `1`: можно начинать быстро, мало документов.
- `2`: простой старт, но нужны базовые проверки.
- `3`: реалистично при подготовке документов, адреса и дохода.
- `4`: нужен спонсор, работодатель, доход, сложный пакет или долгий срок.
- `5`: высокий порог денег/квалификации/квот/политического риска.

## Грузия

Статус: `reviewed` для короткого въезда, `needs_review` для работы, бизнеса и ВНЖ.

| Ветка         | Сложность | Как моделировать                                                                                                |
| ------------- | --------: | --------------------------------------------------------------------------------------------------------------- |
| `short_entry` |         2 | Длинный безвизовый старт для разведки, удаленной работы без местного рынка и семейной адаптации.                |
| `medium_stay` |       2-3 | Подходит для 3-12 месяцев, но нужно отдельно проверять налоги, банк, школу, работу и право на деятельность.     |
| `residence`   |       3-4 | Временное проживание через работу, ИТ, недвижимость, инвестиции, семью или учебу; держать как условный маршрут. |
| `remote_work` |         3 | Разделять иностранного работодателя, местного работодателя и самозанятость.                                     |
| `local_work`  |         4 | Проверять новые правила разрешения на работу для employed/self-employed иностранцев без ПМЖ.                    |
| `business`    |       3-4 | ИП/компания, налоговая регистрация, право на предпринимательскую деятельность, возможный ИТ-маршрут.            |
| `family`      |       2-3 | Для короткого въезда легко, для статуса нужны документы родства и основание главного заявителя.                 |
| `study`       |         3 | Проверять учебное зачисление и study residence.                                                                 |
| `pets`        |         3 | Ветпаспорт, чип, вакцина, правила ввоза.                                                                        |

Анкета: ИТ-профиль, грузинский работодатель, самозанятость, бизнес/ИП, недвижимость/инвестиции, семья, питомец, школа, налоговая цель.

Blockers: пользователь хочет работать на местном рынке без права на работу; нет основания для ВНЖ; нет адреса/дохода; семья без переводов.

Unlocks: иностранный удаленный доход, грузинский оффер, регистрация бизнеса, ИТ-профиль, семейная связь, учеба.

Источники проверять: Matsne visa-free ordinance, SDA residence permits, State Employment Support Agency, Revenue Service, Geostat, ветеринарная служба.

## Армения

Статус: `reviewed` для въезда, `needs_review` для процедур работы и e-gov подачи.

| Ветка         | Сложность | Как моделировать                                                                                       |
| ------------- | --------: | ------------------------------------------------------------------------------------------------------ |
| `short_entry` |         1 | Быстрый въезд и адаптация.                                                                             |
| `medium_stay` |         2 | Проверять лимит пребывания в году и дату въезда.                                                       |
| `residence`   |         3 | Временное проживание через работу, бизнес, учебу, семью, происхождение и другие основания.             |
| `remote_work` |       2-3 | Если нет местной занятости, основной риск бытовой/налоговый; при местной работе нужна отдельная ветка. |
| `local_work`  |         3 | Employment-based status, работодатель и документы.                                                     |
| `business`    |         3 | Бизнес-активность как возможное основание, но нужны реальные подтверждения.                            |
| `family`      |       2-3 | Документы родства, главный заявитель, дети и школа.                                                    |
| `study`       |         3 | Зачисление открывает временное проживание.                                                             |
| `pets`        |         3 | Ветдокументы и правила ввоза.                                                                          |

Анкета: сколько дней уже были в Армении, работодатель в Армении, бизнес-активность, армянское происхождение/родство, учеба, готовность к нотариальному переводу паспорта и медсправке.

Blockers: исчерпан лимит пребывания, нет основания для резидентства, нет перевода паспорта/медсправки.

Unlocks: трудовое основание, бизнес-документы, зачисление, семья/родство, подготовленные переводы.

Источники проверять: MFA Armenia, Migration and Citizenship Service, migration.e-gov.am, work permits, госреестр бизнеса, образование, ветеринария.

## Как переносить в `data/`

1. Каждая строка ветки становится отдельным `Route` или подветкой внутри маршрута страны.
2. `short_entry` и `medium_stay` не должны маскировать отсутствие долгого основания.
3. Для стран с высоким политическим/визовым риском (`PL`, `FI`, `CZ`, `LT`, `LV`) короткий въезд по умолчанию должен быть `conditional`, `unknown` или `blocked`, пока анкета не подтверждает допустимую цель, паспорт и исключение.
4. Для стран с удаленной работой (`TR`, `AE`, `CY`, `ES`, `PT`, `HU`, `IT`, `GR`, `BR`, `AR`, `ID`, возможно `TH`) разделять:
   - иностранный работодатель;
   - иностранные клиенты;
   - местный работодатель;
   - работа на местный рынок.
5. Для каждого маршрута нужны `sourceIds`, `lastReviewedAt`, `confidence`, `publicationStatus`.
6. Пользовательский текст должен говорить "может потребоваться" и "решение принимает компетентный орган", а не обещать одобрение.
